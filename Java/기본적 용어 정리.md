# 기본적 용어 정리 
## api
**ApplicationProgrammingInterface**  
한 프로그램에서 다른 프로그램으로 데이터 주고받기 위한 방법  
메뉴판 같은거다.  
ex)웹툰 서비스API를 만들었다고 치면
>1.신과함께  
>2.이말년어쩌구  
>3.헬퍼  
>이렇게 만들어두면  
>사용자:3번 보여주세요
서비스를 하기위해서 미리만들어두는 메뉴판

자바를 예로 들면 
```java
public String stringAPI(String 결과){//이 부분이 API임
  console.log(결과);
  return 결과;          //이 코드를 실행해주세요~
}
```
